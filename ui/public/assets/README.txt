Public site image assets
=========================

Drop real brand assets here (served at /assets/...):

  hero.jpg   -> full-bleed homepage hero photo (referenced by the .hero CSS).
               Until present, the hero shows a green gradient fallback.
  logo.png   -> optional raster logo if you prefer it over the inline SVG mark
               in the page header.

Photos pulled by scripts/scrape_site.py land in scraped/images/ (git-ignored).
Copy the ones you want to use into this folder and commit them.
