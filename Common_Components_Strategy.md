# COMMON COMPONENTS STRATEGY
## Standardized Parts for All Configurations (Straight, T, X, L)

**Goal**: Reduce complexity by building every greenhouse from a small set of shared components.

---

## 1. STANDARD COMPONENT LIST (Core Building Blocks)

| Component              | Description                          | Used In          | Stock Unit |
|------------------------|--------------------------------------|------------------|------------|
| **WRW Bay**            | 48" × 60" Wall-Roof Weldment         | All shapes       | Each       |
| **Gable Set**          | UGW + LGW pair                       | All shapes       | Set        |
| **WRUB Brace**         | Upper Roof Brace (51.5°)             | All shapes       | Each       |
| **WRKB Brace**         | Knee Brace (64.5°)                   | All shapes       | Each       |
| **Reinforced Side Brace** | Lateral stability brace           | All shapes       | Set        |
| **Tee Junction**       | Reinforced connection for T-shape    | T-Shape only     | Each       |
| **Cross Junction**     | Reinforced connection for X-shape    | X-Shape only     | Each       |
| **L Junction**         | Corner connection for L-shape        | L-Shape only     | Each       |
| **Dormer Kit**         | Optional roof feature                | All shapes       | Kit        |
| **Door Kit**           | Optional door assembly               | All shapes       | Kit        |

---

## 2. HOW CONFIGURATIONS ARE BUILT

### Straight Rectangle
- **Formula**: `N × WRW Bays + 2 × Gable Sets + Bracing`
- Example (16 ft): 4 WRW + 2 Gables + 4 Bracing Sets

### T-Shape
- **Formula**: `(Main Bays + Branch Bays) × WRW + 3 × Gable Sets + 1 × Tee Junction + Bracing`
- Example (16 ft main + 8 ft branch): 6 WRW + 3 Gables + 1 Tee + 5 Bracing Sets

### X-Shape
- **Formula**: `(Main Bays + 2 × Branch Bays) × WRW + 4 × Gable Sets + 1 × Cross Junction + Bracing`
- Example (12 ft main + 8 ft cross): 5 WRW + 4 Gables + 1 Cross + 4 Bracing Sets

### L-Shape
- **Formula**: `(Main Bays + Branch Bays) × WRW + 3 × Gable Sets + 1 × L Junction + Bracing`

---

## 3. BENEFITS OF THIS APPROACH

- **Fewer SKUs** to manufacture and stock
- **Faster production** (batch common components)
- **Easier inventory management**
- **Simpler for co-packer**
- **Scalable** — new shapes only need new junction pieces
- **Lower cost** and complexity long-term

---

## 4. RECOMMENDED STOCK STRATEGY

**High-Volume Common Components** (stock in bulk):
- WRW Bays
- Gable Sets
- WRUB, WRKB, and Reinforced Side Braces
- Hardware (Toggle Clamps + H-Bolts)

**Lower-Volume / Shape-Specific** (build to order or small stock):
- Tee Junctions
- Cross Junctions
- L Junctions
- Dormer Kits (optional)

---

## 5. IMPACT ON SYSTEM

The **Configuration Engine** and **Self-Running System** have been updated to use this common component model. All future submittals, BOMs, and production plans will follow this structure.

This is the cleanest, most scalable way to run the business.
