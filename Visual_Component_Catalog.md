# VISUAL COMPONENT CATALOG
## Standard Parts for Modular Greenhouses

**All components are based on existing drawings from the original design package.**

---

## 1. WRW Core Bay (Wall-Roof Weldment)
- **Drawing Reference**: WRW.PDF (Sheets 1-3)
- **Material**: 6061-T6, 1" sq × .065" wall
- **Size**: 48.03125" wide × 60.03125" tall
- **Weight**: 12.5 lbs
- **Key Features**:
  - 5 tubes per assembly
  - 17× 9/64" + multiple 3/8" holes
  - 45° miters on corners
- **Used In**: All shapes (Straight, T, X, L)

---

## 2. Gable Set (UGW + LGW)
- **Upper Gable (UGW)**: Drawing UGW.PDF
  - 61.97" sloped @ 39°/51°
  - Weight: 9.8 lbs
- **Lower Gable (LGW)**: Drawing LGW.PDF
  - 60" verticals + 22.25" horizontals
  - Weight: 11.2 lbs
- **Combined**: 21 lbs per set
- **Used In**: All shapes (2–4 sets depending on configuration)
